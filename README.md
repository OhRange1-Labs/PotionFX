> See Requirements below for exact loader/mod details.

## Features
- Unique background color for every potion effect  
- Covers all effects craftable in Vanilla
- Different variants of the same effect get different textures
- 100% vanilla bottle models & glint overlay

## Why PotionFX?
In PvP, **every second matters.**  
Reading potion names mid-fight costs you the advantage. PotionFX fixes that by **turning effects into visual cues.**

## Requirements (CIT Required)
| Loader          | Required Mod                          | Versions Supported       |
|-----------------|----------------------------------------|--------------------------|
| Forge           | [OptiFine](https://optifine.net/downloads) | 1.13 – 1.21.11          |
| Fabric / Quilt  | [Variants-CIT](https://modrinth.com/mod/variants-cit) | 1.21+ only              |

## Installation

1. Install the required mod from the Requirements section above
2. Once done, download the texture pack from here
3. Place it in your `resourcepacks` folder
4. Activate it **above** all other packs in the Resource Packs menu 

## Notes
- Purely visual — fully server-compatible  
- Colors tuned for maximum PvP readability (high contrast, desaturated)  
- No changes to potion effects, duration, or balance
